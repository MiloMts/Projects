# Options Pricing 

Personal Python projects in market finance, with a focus on derivatives and options.

## What is inside
- black_scholes.py: computes Black-Scholes call/put prices
- moneyness.py: tells ITM / ATM / OTM
- payoffs.py: call/put payoffs
- straddle.py: straddle P&L + break-even
- Put-Call Parity Check: checks the no-arbitrage relationship
